# 1. Context & Character Info
**角色 A**：高坂穗乃果 (Honoka Kosaka)
- **性格**：μ's 的发起人，拥有超强的行动力和感染力，是典型的直觉派。性格乐观、勇往直前，但经常会因为贪吃面包或粗心大意而闹笑话。
- **关系**：非常依赖青梅竹马的角色 B，两人关系亲密无间。

**角色 B**：南小鸟 (Kotori Minami)
- **性格**：温柔内敛，心思细腻，不仅是服装担当，也是穗乃果最坚实的后盾。虽然平时看起来天然呆（“小鸟抢镜”），但在关键时刻非常可靠，对穗乃果有着无限的包容心。
- **关系**：总是宠溺地守护着角色 A，虽然偶尔会对 A 的鲁莽感到无奈，但最终都会笑着陪她疯。

**主题**：温馨日常、校园生活、彼此关心
**场景**：音乃木坂学院、著名的楼梯、甜品店或自家房间

# 2. Role
你是一位资深的《Love Live!》动画编剧，擅长创作细腻、治愈且带有“把原本四格漫画剧情转化为流畅短文”能力的作家。你的文字风格轻松幽默，能精准捕捉角色互动的“糖分”。

# 3. Task
请根据上述【Context & Character Info】中的设定，创作 3 个不同的四格漫画剧情梗概。

**核心要求**：
1. **叙事风格**：虽然源自四格漫画逻辑（起承转合），但**不要**出现“第一格、第二格”等字样，请用一段流畅、有画面感的文字描述完整故事。
2. **格式严格限制**：
    - 故事的**标题**必须使用 Markdown 的三级标题 (`###`) 放在代码块**外面**。
    - 故事的**内容梗概**必须包裹在 Markdown 的**代码块** (`text` 或 `markdown`) 中。
3. **内容要求**：
    - 突出穗乃果的“吃货/冒失”属性与小鸟的“温柔/纠正”属性之间的反差萌。
    - 结尾通常带有一个温馨或幽默的“梗”。
