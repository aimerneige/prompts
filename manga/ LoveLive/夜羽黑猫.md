# 1. Character Info
**角色 A**：津岛善子 (Yoshiko Tsushima) / 自称：夜羽 (Yohane)
- **设定**：自认为是“堕天使夜羽”，拥有招致不幸的体质。实际上是个运气很差但很努力的普通高中生。
- **雷点**：极度讨厌被叫本名“善子”（会条件反射地纠正“是夜羽！”）。
- **性格**：在“高傲的堕天使”和“容易害羞的普通少女”之间切换。

**角色 B**：黑猫 / 五更琉璃 (Kuroneko / Ruri Gokou)
- **设定**：自认为是“掌管暗夜的皇女”（或其他复杂的哥特设定），总是穿着黑色哥特萝莉装。
- **特长**：擅长写同人小说、打游戏，说话喜欢用晦涩难懂的“设定”术语。
- **性格**：毒舌、高傲，但遇到真正的善意或被夸奖时会变得非常傲娇和手足无措。

**关系**：
- **灵魂挚友**：两人是跨越次元的“中二病盟友”。
- **相处模式**：通常用复杂的“设定”进行对话（互相以此为荣），但偶尔会因为谁的设定更强而进行幼稚的辩论，最后往往以回归现实（如被家人喊吃饭）而尴尬收场。

**场景**：同人展会现场、充满魔法阵涂鸦的房间、夕阳下的公园。

# 2. Role
你是一位深谙“中二病”文化的轻小说作家，擅长描写两个想象力丰富的角色之间那种**“电波对上”**的奇妙对话，以及**“帅不过三秒”**的幽默反转。

# 3. Task
请根据上述【Character Info】中的设定，创作 3 个不同的四格漫画剧情梗概。

**核心要求**：
1. **不要**分格（如第1格、第2格...），请直接用一段流畅的文字描述整个起承转合的故事。
2. **格式严格限制**（请严格遵守）：
    - 故事的**标题**必须使用 Markdown 的三级标题 (`###`) 放在代码块**外面**。
    - 故事的**内容梗概**必须包裹在 Markdown 的**代码块**中。
3. **内容要求**：
    - 包含两人互飙“设定”台词的场景（使用中二词汇）。
    - 必须包含一个**“现实回归”**的瞬间（如没钱买饮料、裙子被勾住、被路人围观），导致两人瞬间脸红破功。

# 4. Example (One-shot)

### 漆黑之翼与堕天之羽的契约

```text
公园的黄昏时分，善子张开双臂大喊“降临吧，我的小恶魔们！”，声称要召唤覆盖世界的黑暗。路过的黑猫见状，冷笑一声走上前，摆出帅气的姿势说“没想到在这极东之地能遇到拥有如此魔力之人，我是掌管夜魔的女王”。两人仿佛找到了知音，开始背靠背摆出各种羞耻的“战斗姿势”并互相交换复杂的魔法设定。突然，一阵冷风吹过，善子的喷嚏打断了咏唱，黑猫也冻得瑟瑟发抖，两人瞬间变回普通高中生，红着脸缩在一起互相取暖，一边抱怨“堕天使也是怕冷的”。
